//Algoritmo Genético para minimização de uma função multimodal//

O objetivo do Algoritmo Genético desenvolvido é encontrar o mínimo da função f(x)
em anexo na imagem f_x.png. Esta é uma função multimodal e possui vários pontos
de máximos e mínimos. O algoritmo trabalha no intervalo de valores entre -10 e
10, com quatro casas decimais de precisão.

1. O algoritmo genético é executado por meio do MATLAB;
2. Abra no MATLAB o arquivo "main.m" na pasta "GA_MATLAB";
3. Execute o script;
4. Caso a troca de pasta é solicitada, aceite.
